Solution for: Ladder Alley...

The Answer: approx. 1.231....

Unfortunately, this solution is quite unsatisfying, since the value is not "exact" (i.e. it is not written with a finite amount of numbers, optionally making use of a squareroot or values like pi...).

The mentioned approx. value can be obtained as follows (with thanks to Peter Seeviour):
If W is the width of the alley, then W = sqrt(4 - Z**2),
where Z is the solution to (Z/(Z-1))**2 - Z**2 = 5.
Unfortunately, solving this gives the following: If A = (395 + 60 sqrt 39)**1/3
B = 2(A + (25/A) - 7)/3
then Z = 1/2 + (sqrt B + sqrt((24 sqrt 2 / sqrt B) - B -14)) / (2 sqrt 2)
All this gives W = 1.231...
