Divisibility Intrigue

Solution: The number is 2,520.

The first four primes are 2, 3, 5, 7.
The first four composites are 4, 6, 8, 9, or, as prime factors: 2 x 2, 2 x 3, 2 x 2 x 2, and 3 x 3.

So, the number is 2 x 2 x 2 x 3 x 3 x 5 x 7 = 2,520.
